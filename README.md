# GladiatorGame
Gladiator rogue-like developed by Maximo Sanchez. Game is fully developed in Unity C#
The game features:
- Realistic ragdoll physics
- Stat multipliers
- Wave system for enemy spawning
- In-game currency to purchase upgrades
- FPS camera with a Sword (only weapon for now)
- Two enemy types (for now) Barbarians and Archers
